# Mental Health Support Predictor

ML application predicting mental health support needs for 62M tech employees. Built GUI with classification models (KNN, Decision Trees, Logistic Regression, SVC). Developed complete data pipeline with preprocessing, encoding, scaling, and evaluation via F1 scores, confusion matrices, ROC curves. Team of 4 using GitHub for version control.

## Repository Structure

### `/ideas and weekly notes`
Contains project planning documents, meeting notes, and ideation materials:
- Weekly progress updates (Week 2, 3, 4)
- Team collaboration documents
- Project presentations and documentation

### `/implementation`
Contains all project code and resources:
- **`cleaned_data/`** - Processed datasets ready for modeling
- **`eda/`** - Exploratory data analysis notebooks and visualizations
- **`image/`** - Project images and visual assets
- **`input/`** - Raw data files
- **`Group project latest.ipynb`** - Main project notebook with complete pipeline

## Getting Started

1. Clone the repository
2. Review weekly notes in `/ideas and weekly notes` to understand project evolution
3. Navigate to `/implementation` to explore the code
4. Run `Group project latest.ipynb` for the complete analysis
